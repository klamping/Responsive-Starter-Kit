Responsive Starter Kit

Overview

This responsive starter kit is primarily a learning tool. It's a stripped down, but highly functional, toolkit to help you learn about the inner workings of responsive web design. From a practical standpoint, it's also a good starting point to jump-start a typical responsive web design project. Sample HTML files are included to demonstrate a variety of typical page layouts: style guide, left sidebar, right sidebar, full width, audio/video transcript, and a sample form.

There's a video training course that accompanies this starter kit.

Make It Your Own

Have a look around. Take a close look at the source of each HTML page to get started. Load them up in your browser and see how they react at various viewport widths and on various mobile devices. While you're having a look around, be sure to poke around in the CSS files, located in the assets/styles directory. You can also find some empty JavaScript starter files in the assets/scripts directory.

Everything is fair game for editing, changing, and even deleting. This starter kit's main purpose is to get you started down the right path. The theme.css file is for demonstration purposes only—it's not meant to influence design or user experience decisions. You are encouraged to replace it with CSS styles that match your site's design. Feel free to delete it entirely if you'd like.
